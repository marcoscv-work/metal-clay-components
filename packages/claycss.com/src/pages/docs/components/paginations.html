Pagination

Preset pagination styles helps divide up large blocks of content on your site or app. Use class {literal}ellipsis{/literal} or {literal}dropdown{/literal} on a pagination list item to separate pagination controls from page numbers.

Pagination Sizes

More sizing options, add {literal}pagination-sm{/literal} or {literal}pagination-lg{/literal} to any pagination component to make it smaller or larger. It can be added to the parent container, such as {literal}pagination-bar{/literal}, to size all the pagination components that reside within.
