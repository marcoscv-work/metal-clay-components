Tabular List Group

Align content inside a list-group like a table with {literal}.tabular-list-group{/literal}.

{literal}.list-group-item{/literal} behaves as a table row.

Use a combination of {literal}.list-group-item-field{/literal} and {literal}.list-group-item-content{/literal} to control the size of each cell. {literal}.list-group-item-field{/literal} has width: 1% and will expand only as wide as its content as long as {literal}.list-group-item-content{/literal} is present as a sibling.

{literal}.list-group-item-content{/literal} will expand to fill the remaining space.

In IE9 the class {literal}truncate-text{/literal} is unsupported in {literal}tabular-list-groups{/literal} and will break to a new line.

Tabular List Group Unstyled

A less opinionated and simplified version of {literal}tabular-list-group{/literal} to help layout small bits of content inside other components such as sidebar. The component {literal}tabular-list-group-unstyled{/literal} doesn't split into columns; to make tabular-list-group-unstyled split into columns, add {literal}display: table;{/literal} to the class {literal}tabular-list-group-unstyled{/literal} and {literal}display: table-row;{/literal} to {literal}list-group-item{/literal}.

List Group Card

Deprecated {literal}list-group-card{/literal} as of v1.0.9 and will be removed in v2.0.0.

Add class {literal}list-group-card{/literal} to {literal}list-group{/literal} then put a checkbox-card in a {literal}list-group-item{/literal}.

  • List Heading

Hide Checkboxes in Mobile

Deprecated {literal}list-group-card-checkbox-hidden-xs{/literal} as of v1.0.9 and will be removed in v2.0.0.

Add class {literal}list-group-card-checkbox-hidden-xs{/literal} to {literal}list-group-card{/literal} to hide {literal}input[type="checkbox"]{/literal} in mobile, then wrap your icon or user-icon with {literal}list-group-card-icon{/literal} to toggle the check icon.

The check icon can be customized through the variable {literal}$list-group-card-checkbox-hidden-icon{/literal}.
