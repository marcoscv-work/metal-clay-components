Table List

A table styled like a list. The active state can be invoked by adding class {literal}active{/literal} to the {literal}{/literal} element. It is recommended that table-list's columns containing information of lesser importance be hidden in smaller screen sizes with Responsive Utilities due to space constraints.

Class {literal}table-bordered{/literal} not compatible with {literal}table-list{/literal}.

Table Helpers

{literal}table-autofit{/literal} constrains table columns to be only as wide as its content, but must be used with {literal}table-cell-content{/literal}. {literal}table-cell-content{/literal} will fill the remaining space.

Vertical Alignment

We have added some classes to help vertically align contents inside a table. The classes {literal}table-valign-bottom{/literal}, {literal}table-valign-middle{/literal}, and {literal}table-valign-top{/literal} on {literal}{/literal} will vertically align table cell contents on the bottom, middle, and top, respectively.

The classes {literal}thead-valign-bottom{/literal}, {literal}thead-valign-middle{/literal}, and {literal}thead-valign-top{/literal} on {literal}

{/literal} will vertically align the contents inside the table head.

The classes {literal}tbody-valign-bottom{/literal}, {literal}tbody-valign-middle{/literal}, and {literal}tbody-valign-top{/literal} on {literal}

{/literal} will vertically align the contents inside the table body.

table-valign-bottom table-valign-middle table-valign-top
tbody-valign-bottom tbody-valign-middle tbody-valign-top
thead-valign-bottom thead-valign-middle thead-valign-top
