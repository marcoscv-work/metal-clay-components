Nameplates

Quickly layout user details with the nameplate component. It uses a table based layout so you can vertically align any content inside with the css property {literal}vertical-align{/literal}.

Use a combination of {literal}.nameplate-field{/literal} and {literal}.nameplate-content{/literal} to control the size of each cell. {literal}.nameplate-field{/literal} has width: 1% and will expand only as wide as its content as long as {literal}.nameplate-content{/literal} is present as a sibling.

Nameplate Label Autofit Xs Max

Add class {literal}nameplate-label-autofit-xs-max{/literal} to the container or to a single nameplate-label to make them fill the width of the container in screen sizes 767px and below.

Use the Sass mixin {literal}nameplate-label-autofit($media-feature){/literal} to create custom nameplate-label-autofit breakpoints e.g. {literal}.nameplate-label-autofit-sm { @include nameplate-label-autofit('min-width: $screen-sm'); }{/literal} for screen sizes 768px and up.
