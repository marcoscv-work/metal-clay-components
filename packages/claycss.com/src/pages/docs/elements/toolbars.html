Toolbar

Quickly prototype a toolbar for your portlet, control panel, or site with this component and easily adjust the height of your toolbar while maintaining vertical alignment. Adjust the height of your toolbar by changing the height of {literal}.toolbar-group{/literal}, {literal}.toolbar-group-right{/literal}, and {literal}.toolbar-group-expand{/literal}.

In IE9 you will need to adjust the top css property of dropdown-menu to match the height of your toolbar.

How to use toolbar-group-content and toolbar-group-field

Use only {literal}toolbar-group-content{/literal} inside {literal}toolbar-group{/literal} if you want to create a group that is only as wide as the content inside.

Use only {literal}toolbar-group-field{/literal} inside {literal}toolbar-group{/literal} if you want to create a group that is equally spaced across {literal}toolbar-group{/literal}.

Use one {literal}toolbar-group-content{/literal} and the rest with {literal}toolbar-group-field{/literal} inside {literal}toolbar-group{/literal} if you want to create a group with one element that spans the remaining width of {literal}toolbar-group{/literal}.

Use multiple {literal}toolbar-group-content{/literal}s and multiple {literal}toolbar-group-field{/literal}s inside {literal}toolbar-group{/literal} if you want to create a group with multiple items only as wide as its content and the remaining items to share the remaining space inside {literal}toolbar-group{/literal}.

Fill Remaining Space with Form Element

To fill the remaining space with one form element such as a search box, declare {literal}.toolbar-group-expand{/literal} last and use {literal}.toolbar-group-content{/literal}.
